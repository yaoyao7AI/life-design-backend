# 📋 数据库变更后需要提供的信息

## 🎯 如果阿里云 RDS 数据库变更了，请提供以下信息

---

## ✅ 必须提供的信息（核心）

### 1. 数据库实例信息

**实例ID：**
```
例如：rm-2ze0yppdih8t4t4e4
```
- 在阿里云 RDS 控制台可以看到
- 通常在实例列表的第一列

**实例名称：**
```
例如：life-design-db
```
- 这是你给实例起的名字（可选，但有助于识别）

---

### 2. 数据库连接信息

**数据库名称：**
```
例如：life_design
```
- 这是你要操作的数据库名称
- 不是实例名称

**数据库用户名：**
```
例如：root 或 life_design_user
```
- 用于连接数据库的用户名

**数据库密码：**
```
例如：your_password_here
```
- ⚠️ **注意：** 如果只是迁移字段，通常不需要密码
- 如果需要修改后端连接配置，才需要提供

**数据库主机地址：**
```
例如：rm-2ze0yppdih8t4t4e4.mysql.rds.aliyuncs.com
```
- 在 RDS 控制台 → 实例详情 → 连接信息中可以找到
- 格式通常是：`实例ID.mysql.rds.aliyuncs.com`

**数据库端口：**
```
例如：3306
```
- MySQL 默认端口是 `3306`
- 在 RDS 控制台 → 实例详情 → 连接信息中可以找到

---

## 📍 如何找到这些信息？

### 方法 1：通过 RDS 控制台

1. **登录阿里云控制台**
   - 访问：`https://rdsnext.console.aliyun.com`

2. **找到你的实例**
   - 在实例列表中找到实例ID：`rm-2ze0yppdih8t4t4e4`

3. **点击实例，进入详情页**

4. **查看"连接信息"**
   - 在详情页找到"连接信息"或"数据库连接"部分
   - 可以看到：
     - 内网地址（主机地址）
     - 端口
     - 数据库名称列表

5. **查看"账号管理"**
   - 在详情页找到"账号管理"或"数据库账号"
   - 可以看到数据库用户名列表

---

### 方法 2：通过 DMS（数据管理）

1. **登录 DMS**
   - 访问：`https://dms.console.aliyun.com`

2. **选择实例**
   - 在左侧实例列表中选择：`rm-2ze0yppdih8t4t4e4`

3. **查看连接信息**
   - 右键点击实例 → 查看连接信息
   - 或点击实例详情

## ⚠️ 安全提醒

### 🔒 密码安全

**如果只是执行数据库迁移（添加字段）：**
- ✅ **不需要提供密码**
- ✅ 只需要实例ID和数据库名称
- ✅ 通过 DMS 登录即可

**如果需要修改后端连接配置：**
- ⚠️ **需要提供密码**
- ⚠️ **建议通过私密方式提供**（不要公开）
- ⚠️ **修改配置后建议更换密码**

---

## 🎯 常见变更场景

### 场景 1：实例ID变更

**问题：** 创建了新实例，实例ID变了

**需要提供：**
- ✅ 新实例ID
- ✅ 数据库名称（如果没变，可以不说）
- ✅ 其他连接信息（如果变了）

---

### 场景 2：数据库名称变更

**问题：** 数据库名称从 `life_design` 改为 `life_design_new`

**需要提供：**
- ✅ 新数据库名称
- ✅ 实例ID（如果没变，可以不说）

**SQL 需要修改为：**
```sql
USE life_design_new;  -- 改为新名称
ALTER TABLE users ADD COLUMN password VARCHAR(255) NULL AFTER phone;
```

---

### 场景 3：连接地址变更

**问题：** 数据库主机地址变了

**需要提供：**
- ✅ 新主机地址
- ✅ 新端口（如果变了）
- ✅ 用户名和密码（如果需要修改后端配置）

---

### 场景 4：账号变更

**问题：** 数据库用户名或密码改了

**需要提供：**
- ✅ 新用户名
- ✅ 新密码（如果需要修改后端配置）

---
