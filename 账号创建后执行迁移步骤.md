### 步骤 3：查看执行结果（30秒）

**执行后，下方会显示结果：**

#### ✅ 成功标志

**应该看到：**
1. **执行成功的提示**
   - 显示"执行成功"或"Affected rows: 1"

2. **表结构结果**
   - 显示 `users` 表的所有字段
   - 在表格中找到 `password` 字段行
   - 应该显示：
     ```
     Field: password
     Type: varchar(255)
     Null: YES
     Key: (空)
     Default: NULL
     Extra: (空)
     ```

#### ❌ 如果报错

**查看错误信息，常见错误：**

1. **字段已存在**
   ```
   ERROR 1060: Duplicate column name 'password'
   ```
   - ✅ **这是好消息！** 说明字段已经存在
   - 执行 `DESCRIBE users;` 验证字段存在即可

2. **表不存在**
   ```
   ERROR 1146: Table 'life_design.users' doesn't exist
   ```
   - 确认数据库名称正确
   - 执行 `SHOW TABLES;` 查看所有表

3. **权限不足**
   ```
   ERROR 1142: ALTER command denied
   ```
   - 确认使用的是高权限账号
   - 联系数据库管理员授权
