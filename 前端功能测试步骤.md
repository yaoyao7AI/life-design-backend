## 🚨 常见问题排查

### 问题 1：前端无法连接后端

**错误：** `Failed to fetch` 或 `Network Error`

**排查步骤：**
1. 检查后端服务是否运行：
   ```bash
   curl http://123.56.17.118:3000/api/auth/me
   ```
2. 检查浏览器控制台的 `BASE_URL` 是否正确
3. 检查 CORS 配置

---

### 问题 2：验证码发送失败

**错误：** `验证码发送失败`

**排查步骤：**
1. 检查后端日志
2. 检查 SMS 服务配置
3. 检查手机号格式

---

### 问题 3：登录失败

**错误：** `登录失败` 或 `验证码错误`

**排查步骤：**
1. 确认验证码正确（查看后端日志或数据库）
2. 检查验证码是否过期（通常5分钟）
3. 重新获取验证码

---

### 问题 4：修改密码失败

**错误：** `密码修改失败` 或 `密码功能暂未启用`

**排查步骤：**
1. 确认数据库迁移已完成：
   ```bash
   cd /Users/mac/Desktop/life-design-backend
   ./verify_migration.sh
   ```
2. 检查 Token 是否有效
3. 检查后端日志错误信息
