# 🔍 查看日志并测试登录

## 📋 查看所有日志

**在服务器上执行：**

```bash
cd /root/apps/life-design-backend

# 查看最近的日志（包括错误和输出）
pm2 logs life-design-backend --lines 50 --nostream

# 或者只看错误日志
pm2 logs life-design-backend --err --lines 30 --nostream

# 或者只看输出日志
pm2 logs life-design-backend --out --lines 30 --nostream
```
