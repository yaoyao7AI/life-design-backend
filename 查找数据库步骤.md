# 🔍 查找数据库步骤

## 🎯 目标
找到正确的数据库名称，或创建 `life_design` 数据库。

---

## 📋 步骤 1：查看所有数据库

### 在 SQL 控制台执行：

```sql
SHOW DATABASES;
```

### 点击"执行(F8)"按钮

---

## 🔍 步骤 2：查看结果

**执行后，会显示所有数据库列表，类似：**

```
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| life_design        |  ← 如果存在，会在这里
| ...                |
+--------------------+
```

---

## ✅ 情况 1：找到了数据库

**如果列表中看到：**
- `life_design`（带下划线）
- `life-design`（带连字符）
- `lifedesign`（无分隔符）
- 或其他类似名称

**操作：**
1. **记住数据库名称**
2. **切换到该数据库：**
   ```sql
   USE [数据库名称];
   ```
   例如：`USE life_design;` 或 `USE life-design;`

3. **然后执行迁移 SQL**

---

## ❌ 情况 2：没有找到数据库

**如果列表中没有 `life_design` 或类似名称：**

### 选项 A：创建新数据库（推荐）

**执行以下 SQL：**

```sql
CREATE DATABASE life_design CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE life_design;
```

**然后创建 `users` 表：**

```sql
CREATE TABLE users (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  phone VARCHAR(20) NOT NULL UNIQUE,
  password VARCHAR(255) NULL,
  nickname VARCHAR(50),
  avatar VARCHAR(255),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

### 选项 B：使用现有数据库

**如果看到其他数据库名称（如 `test`, `app_db` 等）：**

1. **确认是否是你要用的数据库**
2. **切换到该数据库**
3. **检查是否有 `users` 表：**
   ```sql
   USE [数据库名称];
   SHOW TABLES;
   ```

---

## 🔍 步骤 3：检查 users 表是否存在

**切换到数据库后，执行：**

```sql
SHOW TABLES;
```

**应该能看到 `users` 表。**

**如果没有 `users` 表，需要创建：**

```sql
CREATE TABLE users (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  phone VARCHAR(20) NOT NULL UNIQUE,
  password VARCHAR(255) NULL,
  nickname VARCHAR(50),
  avatar VARCHAR(255),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 📋 完整操作流程

### 1. 查看所有数据库

```sql
SHOW DATABASES;
```

### 2. 根据结果选择：

**如果找到数据库：**
```sql
USE [数据库名称];
SHOW TABLES;
```

**如果没有找到：**
```sql
CREATE DATABASE life_design CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE life_design;
CREATE TABLE users (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  phone VARCHAR(20) NOT NULL UNIQUE,
  password VARCHAR(255) NULL,
  nickname VARCHAR(50),
  avatar VARCHAR(255),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3. 验证表结构

```sql
DESCRIBE users;
```
